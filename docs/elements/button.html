Button Sizes

Add the .mini, .small, .large or .huge class to a button to make it smaller or larger. Also, you can add the .block class for a full-width button.

?
<button class="button mini">Mini</button>
<button class="button small">Small</button>
<button class="button">Default</button>
<button class="button large">Large</button>
<button class="button huge">Huge</button>
 
<button class="button block">Block Button</button>
